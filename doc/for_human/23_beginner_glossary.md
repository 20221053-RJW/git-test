# 초보자 용어 사전

> **관련:** `doc/for_agent/18_glossary.md`

| 용어 | 쉬운 뜻 |
|------|---------|
| **프론트엔드** | 사용자가 보는 화면 |
| **백엔드** | 서버·저장·로그인 처리 |
| **API** | 데이터 주문 메뉴 |
| **DB / 데이터베이스** | 데이터 창고 (표) |
| **Supabase** | DB + API를 제공하는 클라우드 |
| **Firebase** | 구글이 제공, 로그인 등 |
| **React** | 화면 만드는 JS 라이브러리 |
| **컴포넌트** | 화면 조각 (버튼, 카드) |
| **라우트 / URL** | 주소 (`/app/teams`) |
| **supabase-api.ts** | Supabase CRUD 메뉴판 (`api` 객체) |
| **ai-report.ts** | 마이페이지 리포트 집계 |
| **배포** | 인터넷에 공개 |
| **환경 변수 (.env)** | 비밀번호·키 보관 파일 |
| **RLS** | DB에서 “누가 뭘 볼 수 있는지” 규칙 |
| **JWT / 토큰** | 로그인 증명서 |
| **CI/CD** | 테스트·배포 자동화 |
| **E2E 테스트** | 로봇이 브라우저 조작해 검사 |
| **Alpha** | 데모는 되나 실서비스 전 단계 |
| **Beta** | 실제 데이터로 시험 운영 |
| **MVP** | 최소 기능 시제품 |
| **기술 부채** | 나중에 갚아야 할 설계 타협 |
| **인수인계** | 다음 담당자에게 넘기기 |
| **Figma** | UI 디자인 도구 |
| **Shadcn / Radix** | 예쁜 UI 부품 모음 |
| **TypeScript** | 타입이 있는 JavaScript |
| **Vite** | 빠른 개발·빌드 도구 |
| **pnpm** | 패키지 설치 도구 (npm 대안) |
| **CampusConnect** | 이 프로젝트 서비스 이름 |
| **vision.md** | 프로젝트 철학 원본 |
| **doc/starter.txt** | AI 온보딩 한 장 요약 |

더 많은 용어: `doc/for_agent/18_glossary.md`

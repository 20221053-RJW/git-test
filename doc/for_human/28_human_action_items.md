## AI가 세션 시작·종료할 때 하는 확인

1. 이 파일을 연다.
2. `미완료` 표에서 **인간 확인 가능한 항목**을 점검한다 (`.env` 키 존재, 사용자 “완료” 발언 등).
3. 완료된 항목은 **완료한 일** 표로 옮기고 `완료 시각` 기록.
4. 새로 막힌 일이 생기면 **미완료**에 행 추가 (ID `H-00x` 증가).
5. `25_ai_work_log.md` 맨 위에 오늘 AI 작업 기록 (시분초).

**AI는 미완료 항목이 있어도** 블로커가 아닌 다른 TODO·코드·문서 작업을 **계속** 할 수 있습니다.

# CampusConnect 구현 요약

## 최근 업데이트 (2026-05-06)

### 새로 추가된 페이지

#### 1. 팀 목록 페이지 (`/app/teams`)
- **주요 기능:**
  - 1조~5조 팀 카드 그리드 (5열 레이아웃)
  - 팀별 배지 표시 ("🔥 🚀 가장 조회수가 높은 조")
  - 팀장, 팀원 아바타, 진행률 바, 주간 리포트
  - 공지사항 섹션
  - "새 팀 만들기" 버튼

- **모달 컴포넌트:**
  1. **팀 상세 모달**: 팀 카드 클릭 시 표시
     - 프로젝트 제목 (예: "캠퍼스 카풀 웹서비스")
     - 팀원 아바타
     - "입장하기" 버튼 → 팀 상세 페이지로 이동
     - 진행률 표시 (50% 진행중)
     - 작업 체크리스트 (아이디어 기획, 디자인 작업, 퍼블리싱 작업, 기능 구현)
     - 주간 보고서 안내
     - "통계 보기" 버튼

  2. **통계 모달**: 팀 상세 모달에서 "통계 보기" 클릭 시
     - 주차별 통계 (1주차~5주차)
     - 각 주차별 커밋, 파일, 라인 수
     - 전체 요약 (총 커밋 224, 총 파일 69, 총 라인 10,540)

  3. **주목받는 프로젝트 모달**: 하단 버튼 클릭 시
     - 실시간 주목받는 프로젝트 목록
     - 프로젝트 제목, 설명, 시간

#### 2. 팀 상세 페이지 (`/app/teams/:id`)
- **레이아웃:** 3열 그리드 (1:2 비율)

- **왼쪽 패널:**
  - 파일 트리 (폴더/파일 구조)
  - 코드 실행 버튼
  - 문제 발견 버튼
  - 파일별 이야기 (채팅) 버튼
  - 프로젝트 미리보기 섹션

- **오른쪽 패널:**
  - AI 챗봇 안내 배너
  - 프로젝트 실행 미리보기
  - 주차별 진행률 차트 (1주차~5주차, 컬러 바)
  - 팀원 목록 및 기여도
  - 팀원 평가 버튼

- **모달 컴포넌트:**
  1. **채팅 모달**: 
     - 채팅방 목록 (의명, 성보경 교수님)
     - 메시지 입력 영역
     
  2. **평가 모달**:
     - 개별 팀원 평가 (윤기영, 류지원, 최지연, 김승수)
     - 평균 기여도 표시 (90%, 85%, 95%, 10%)
     - 작업 완성도, 문제 해결력, 총체적 평가 입력

### 네비게이션 업데이트

**이전:**
- 수업, 프로필, 프로젝트, Q&A, 팀 생성

**현재:**
- **수업** (`/app/courses`)
- **수강자들** (`/app/profile`)
- **팀** (`/app/teams`)

**스타일 변경:**
- 배경색: 그라데이션 → 순수 검정 (`bg-black`)
- 활성 탭 색상: 파란색 (#3676ff)
- 비활성 탭: 회색 → 흰색 호버

### 라우팅 구조

```
/app/teams                    → TeamsPage (팀 목록)
/app/teams/:id               → TeamDetailPage (팀 상세)
```

### 디자인 시스템

**색상:**
- Primary Blue: #3676ff, #155dfc, #1962ff
- Blue Gradient: from-blue-600 to-blue-800
- Background: bg-gray-50
- Cards: bg-white

**컴포넌트:**
- 팀 카드: 그라데이션 헤더 (1조 배지용), 흰색 바디
- 모달: rounded-xl, shadow-2xl
- 버튼: rounded-lg, 호버 효과
- 진행률 바: h-2 (작은 바), h-3 (큰 바)

### 주요 개선사항

1. **확장성**: 5개 팀 카드를 동적으로 렌더링
2. **모달 관리**: 3개의 독립적인 모달 (팀 상세, 통계, 주목 프로젝트)
3. **네비게이션 간소화**: 3개 탭으로 정리
4. **반응형**: 그리드 시스템 사용 (md, lg, xl 브레이크포인트)
5. **상호작용**: 클릭, 호버 효과, 전환 애니메이션
